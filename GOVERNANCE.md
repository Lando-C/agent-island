# Governance

Agent Island is currently maintained by the repository owner.

## Decision Principles

1. State accuracy before animation.
2. Safe human handoff before automatic approval.
3. Exact jump targets before generic app activation.
4. Diagnostics before guesswork.
5. Optional local integrations stay optional.

## Maintainer Responsibilities

- Review security-sensitive changes carefully.
- Keep third-party license notices accurate.
- Reject changes that silently approve dangerous tools.
- Keep release notes honest about what is implemented vs planned.

## Project Status

Agent Island is pre-1.0. APIs, hooks, and UI structure may change while the
product core stabilizes.
